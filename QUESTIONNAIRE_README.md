# ESG 调查问卷配置

## 1. 环境政策

- `policy_options` (多选)
  - 问题：贵公司是否有关于以下环境议题的正式政策？
  - 选项：
    - 能源消耗与温室气体 (GHG)
    - 水资源
    - 大气污染 (非温室气体)
    - 材料、化学品与废弃物
    - 生物多样性
    - 产品使用寿命终止 (如回收)

- `quantitative_target` (文本)
  - 问题：政策中是否包含定量目标？(需提供目标数值与年份)

---

## 2. 减排与废弃物措施

- `energy_measures` (多行文本)
  - 问题：在减少能源消耗和温室气体排放方面，采取了哪些措施？

- `waste_measures` (多行文本)
  - 问题：在废弃物与化学品管理方面，采取了哪些措施？

---

## 3. 关键绩效指标 (KPIs)

- `scope1` (数值)
  - 问题：Scope 1 (直接排放) 的数值是多少？(单位：吨 CO2 当量)

- `scope2` (数值)
  - 问题：Scope 2 (能源间接排放) 的数值是多少？(单位：吨 CO2 当量)

- `scope3` (数值)
  - 问题：Scope 3 (上下游其他间接排放) 的数值是多少？(单位：吨 CO2 当量)

- `energy_total` (数值)
  - 问题：总能耗是多少？(单位：kWh)

- `renewable_ratio` (百分比)
  - 问题：可再生能源占比是多少？(0-100)

- `hazardous_waste` (数值)
  - 问题：有害废弃物总量是多少？(单位：kg)

- `nonhazardous_waste` (数值)
  - 问题：无害废弃物总量是多少？(单位：kg)

---

## 4. 碳管理实践

- `ghg_practice` (多选)
  - 问题：关于 GHG 监测和报告实践，以下哪些适用？
  - 选项：
    - 排放核算符合 ISO 14064-1 或 GHG Protocol 标准
    - 排放数据经过第三方验证 (ISAE 3410 等)
    - 报告已向公众披露

- `carbon_target` (多选)
  - 问题：关于碳减排目标，以下哪些适用？
  - 选项：
    - 已公开承诺科学碳目标 (SBTi)
    - 已有经 SBTi 批准的减排目标
    - 设有年度减排目标达成进度的审查机制
